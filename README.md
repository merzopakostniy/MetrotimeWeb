# MetrotimeWeb

Mobile-first web/PWA prototype for Metrotime.

## Local Start

```bash
npm start
```

Open `http://localhost:4173`.

## Free Hosting

Static hosting works without a build step:

- Vercel: import this folder/repository and keep build command empty.
- Netlify: deploy the folder with publish directory `.`.
- GitHub Pages: publish from the repository root.
