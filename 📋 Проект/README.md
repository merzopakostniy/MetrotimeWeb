# MetrotimeWeb — Mobile Web Version

## Цель
Мобильная веб-версия iOS-приложения Metrotime. Доступна всем без App Store + встроенная оплата подписок (Premium).

## Статус
🟡 В разработке — старт: 2026-04-26

## Стек (предварительно)
- Frontend: React / Next.js (PWA)
- Backend: Node.js / Supabase
- Платежи: Stripe / YooKassa
- Хостинг: Vercel / Railway

## Ключевые отличия от iOS-версии
- Доступ без App Store (любой браузер)
- Подписка через веб (без комиссии Apple 30%)
- PWA — можно добавить на главный экран
