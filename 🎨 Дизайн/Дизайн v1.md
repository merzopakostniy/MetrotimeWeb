# Дизайн v1

Дата: 2026-04-26

## Цель

Собрать первый мобильный web-интерфейс МетроТайм как рабочий продуктовый прототип, а не промо-страницу.

## Принципы

- Первый экран - приложение, не лендинг.
- Mobile-first, но нормально выглядит на desktop.
- Плотный рабочий интерфейс: смены, деньги, часы, норма, быстрые действия.
- Нижняя навигация как в мобильном приложении.
- Premium виден, но не мешает базовому учету смен.
- Палитра спокойная и контрастная, без одноцветной "синей" или "фиолетовой" темы.

## Первый набор экранов

- Сегодня.
- Смены.
- Норма.
- Статистика.
- Premium.

## Визуальный язык

- Темный графитовый фон.
- Акцент: зеленый/бирюзовый для статуса смены и нормы.
- Дополнительный акцент: янтарный для денег и Premium.
- Карточки с малым радиусом, рабочие и компактные.
- Флаги смен - маленькие бейджи.
- Время и деньги - главные числа.

## Деплой

Первый прототип статический:
- `index.html`;
- `styles.css`;
- `app.js`;
- `manifest.json`.

Бесплатные варианты:
- Vercel static deploy;
- Netlify static deploy;
- GitHub Pages.
